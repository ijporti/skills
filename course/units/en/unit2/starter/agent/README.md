# Unit 2 Agent Starter

This directory is the runnable companion for the "Inside the Box" section of Unit 2.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Set `ANTHROPIC_API_KEY`, then run:

```bash
python app.py
```

`app.py` intentionally leaves `select_skills()` unfinished. Implement it yourself, then compare with `solution_select_skills.py`.
