# Hugging Face Brand — Gradio

When building Gradio demos, apply Hugging Face's visual identity.

## Colors

- Primary yellow: `#FFD21E`
- Secondary orange: `#FF9D00`
- Text gray: `#6B7280`

## Typography

- Font: Source Sans Pro
- Headings: bold, sentence case
- Body: comfortable, welcoming copy

## Layout

- Center the app
- Use breathing room between sections
- Prefer examples as visible cards instead of burying them in advanced widgets
